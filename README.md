# Aries Theme Kit

The Aries Theme Kit teaches Claude Code how to build UIs that match the Aries platform. Install it once and Claude automatically uses Aries colors, layout patterns, and components whenever you build something new.

## Install

Run this command in your terminal:

```
git clone git@github.com:generaitve-team/aries-theme-kit.git ~/.claude/skills/aries-theme
```

That's it. Claude Code will detect the skill automatically the next time you start a conversation.

## Update

To get the latest version, run:

```
cd ~/.claude/skills/aries-theme && git pull
```

## How to Use

Just talk to Claude like normal. When you ask Claude to build a UI, create a page, or set up a new project, it will automatically use Aries design conventions. You can also type `/aries-theme` to explicitly activate it.

## What's Included

- **Theme colors** -- the full Aries color palette (navy, teal, warm grays) ready to use in any project
- **Layout patterns** -- dashboard cards, data tables, form layouts, detail pages, and more
- **27 pre-selected UI components** -- buttons, inputs, dialogs, tables, and other building blocks from shadcn/ui
- **Typography system** -- Inter font family with consistent sizing and weight conventions
